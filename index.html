GitHub Save / Load

This keeps the game code secret-free. Paste a fine-grained GitHub token here only in your own browser. The game stores it locally, then saves progress to saves/player-save.json in this private repo.

Token permissions: fine-grained token, this repository only, Contents: Read and Write. The token is never committed to the repo.
